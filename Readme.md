## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install pipenv package

```bash
  pip install pipenv
```

Install dependencies

```bash
  pipenv install -r requirements.txt
```

Start the server

```bash
  py manage.py runserver
```
